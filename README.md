# PetFlow API

API REST desenvolvida para gerenciamento de pet shop, permitindo o controle de clientes, pets, funcionários e serviços realizados.

O projeto está sendo desenvolvido com foco em boas práticas de arquitetura backend, organização de código, autenticação segura e escalabilidade, simulando um sistema real de mercado.

## 🚀 Tecnologias Utilizadas

* Java 21
* Spring Boot
* Spring Security
* JWT Authentication
* PostgreSQL
* Spring Data JPA
* Lombok
* Maven
* Swagger/OpenAPI
* Docker

---

## 📌 Funcionalidades

* Cadastro de clientes
* Cadastro de pets
* Cadastro de funcionários
* Controle de serviços
* Catálogo de serviços
* Relacionamento entre clientes e pets
* Autenticação com JWT
* Validação de dados
* Documentação da API

## 🤝 Contribuição

Contribuições são bem-vindas.

Para contribuir:

1. Faça um fork do projeto
2. Crie uma branch:

```bash
git checkout -b feature/minha-feature
```

3. Faça suas alterações
4. Commit suas mudanças:

```bash
git commit -m "feat: minha nova feature"
```

5. Envie para sua branch:

```bash
git push origin feature/minha-feature
```

6. Abra um Pull Request

---

## 📄 Convenção de Commits

Padrão utilizado:

```txt

```
